# User Feedback Form

An interactive client-side feedback form built with vanilla JavaScript, 
HTML5, and CSS3.

## Features
- Real-time form validation with inline error messages
- Live character counter with color warnings (400+ orange, 490+ red)
- Animated speech bubble tooltips on hover
- Email quick-fill buttons with smart cursor placement
- Dynamic feedback catalog with timestamps and delete functionality
- Event delegation and bubbling architecture
- Particle effects animation on form submission

## Technologies
HTML5 | CSS3 | Vanilla JavaScript

## What I Learned
- DOM manipulation and event-driven programming
- Event bubbling, delegation, and stopPropagation()
- CSS animations and custom design systems
- AI-assisted development using Claude (Anthropic)
